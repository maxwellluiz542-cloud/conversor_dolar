# 💱 Conversor de Dólar para Real

Projeto simples em Java que realiza a conversão de dólar para real.

## 📌 Funcionalidades
- Informar o valor do dólar
- Informar a quantidade de dólares
- Calcular o valor total em reais

## 🛠 Tecnologias utilizadas
- Java
- IntelliJ IDEA
